# Xtrivia
Casual trivia game for when you need to test your knowledge or...you're just bored.


## Libraries Used
1. [Retrofit](https://github.com/square/retrofit)
2. [RealmDB](https://realm.io)


## Data Source (JSON)
[Open Trivia DB](https://opentdb.com/api.php)
